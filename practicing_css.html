CSS stands for Cascading Style Sheets and it is used to format the layout of a webpage. CSS allows you to control color, font, size of text, and spacing. The 3 ways to write CSS styles are Incline, Internal, and External. With incline you use style attribute inside HTML elments. In internal, you use style element in the head section. Finally with external use the link element to link an external CSS file.
